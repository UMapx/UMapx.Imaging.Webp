# Installation
Install **UMapx.Imaging.Webp** to your project using [NuGet](https://www.nuget.org/packages/UMapx.Imaging.Webp/) package manager.

```c#
using UMapx.Imaging;
```
To get started with **UMapx.Imaging.Webp** try simple [example](examples).
